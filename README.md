# DEVELOPER PORTFOLIO — README.md
A bold, immersive, animation-rich portfolio built with React + Vite + Tailwind CSS.
This portfolio uses a luxury UI theme, cinematic animations, glassmorphism, and 3D visuals.

No testimonials. No blog. Fully responsive. Smooth. Modern. Senior-level UI/UX.

===============================================================
🚀 PROJECT FEATURES
===============================================================

1) MODERN UI/UX DESIGN
   • High-contrast luxury palette (no blue, no gold)
   • Glassmorphism cards with blurred backgrounds
   • Angled gradient sections
   • 3D depth layers + parallax movement
   • Responsive layout across all devices
   • Fully keyboard accessible

2) ADVANCED ANIMATIONS
   • Framer Motion micro-interactions
       - Button hover elasticity
       - Hero entrance animations
       - Section fade-ins and slide-ins
   • AOS.js scroll-triggered reveals
   • Three.js / R3F 3D elements
   • Custom cursor with loading spinner
   • Cinematic preloader:
         L O A D I N G  (silver fade sequence)

3) HERO SECTION (IMAGE ON RIGHT SIDE)
   • Left: typewriter text + headline + description

   • Parallax hero background drift
   • Buttons:
         - View Projects (scroll)

   • Ripple + glow hover states

4) PROJECTS SECTION (LEFT IMAGE + RIGHT DESCRIPTION)
   • Horizontal project cards:

   • Fully keyboard accessible

6) EXPERIENCE TIMELINE
   • Vertical alternating cards (left/right)

7) CERTIFICATIONS SECTION
   • Flip-on-hover cards

🛠 TECH STACK
===============================================================

Frontend:
   - React + Vite
   - Tailwind CSS (darkMode: "class")

Animations:
   - Framer Motion
   - AOS.js
   - React Scroll
   - Swiper.js (manual + swipe navigation)
   - Three.js / R3F (optional 3D effects)

UI:
   - Lucide Icons
   - Glassmorphism cards
   - Parallax backgrounds

Performance:
   - Lazy-loaded images
   - Optimized animations
   - Lighthouse-friendly

Accessibility:
   - Semantic HTML5
   - Focus rings
   - ARIA labels
   - Keyboard navigation
